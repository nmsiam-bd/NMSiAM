# IUB CGPA Calculator

An accurate and user-friendly CGPA calculator for **Independent University, Bangladesh (IUB)**.  
Calculate semester-wise GPA, cumulative CGPA, export results, and share easily.

---

## 🚀 Features

- Calculate GPA & CGPA
- Support for IUB grading system
- Add multiple semesters
- Export report as: **PDF, Text, CSV, Print**
- Clean UI & Fast calculation
- Mobile responsive
